# Canvas Notes project

---

A web based minimalistic note-taking application with Flask backend and JavaScript frontend

---

## Features

---

- Create and manage notes on the canvas

- Customize note appearance (color, font size)

- Resize notes via corner handles

- Control note layering (bring to front, send to back)

- Copy, paste, and delete notes

- Save notes for later sessions

- Toggle theme between light and dark modes

---

## Tech Stack

- Backend: Flask, Python
- Frontend: HTML, CSS, JavaScript
